    Yurba.js

    About

    Yurba.js is a powerful Node.js library for creating bots and applications that interact with the Yurba.one platform. It provides a comprehensive set of tools for building feature-rich integrations with the Yurba messaging ecosystem.

    Features

    • Real-time Communication: Connect to Yurba.one via WebSocket for instant message processing
    • Message Handling: Send, receive, reply to, and delete messages with ease
    • Command Framework: Register and handle commands with automatic argument parsing
    • Event System: Listen for various events like messages, user actions, and more
    • REST API Integration: Full access to Yurba.one REST API (users, dialogs, posts, etc.)
    • Middleware Support: Add custom middleware for message processing
    • TypeScript Support: Complete TypeScript types for all API objects and methods
    • Promise-based API: Modern async/await interface for all operations

    Installation

    Quick Start

    import { Client } from "yurba.js";

    // Initialize the client with your token
    const client = new Client("YOUR_TOKEN");

    // Listen for the ready event
    client.on("ready", () => {
      console.log("Bot is connected and ready!");
    });

    // Handle incoming messages
    client.on("message", (msg) => {
      console.log(`Received message: ${msg.Text}`);
      
      // Reply to messages containing "hello"
      if (msg.Text.toLowerCase().includes("hello")) {
        msg.reply("Hello there! How can I help you?");
      }
    });

    // Register a command with argument parsing
    client.registerCommand("add", { a: "int", b: "int" }, (message, args) => {
      const sum = args.a + args.b;
      message.reply(`The sum is ${sum}`);
    });

    // Initialize client
    client.init();

    Advanced Usage

    Using Event Listeners

    // Listen for specific events
    client.on("message", (msg) => {
      console.log(`New message from ${msg.User.Tag}: ${msg.Text}`);
    });

    // One-time event listener
    client.once("ready", () => {
      console.log("Bot started successfully!");
    });

    // Wait for specific conditions
    await client.waitFor("message", 
      msg => msg.Text.includes("keyword"), 
      { timeout: 30000 }
    );

    Working with Photos and Attachments

    // Send a message with a photo
    const photoId = "photo123456";
    client.sendMessage(dialogId, "Check out this photo!", null, [photoId]);

    // Get photo information
    const photo = await client.getPhoto(photoId);
    console.log(`Photo URL: ${photo.URL}`);

    Using Middleware

    // Add middleware for message processing
    client.use(async (message) => {
      console.log(`Processing message: ${message.Message?.Text}`);
      // Perform custom logic before message handlers
    });

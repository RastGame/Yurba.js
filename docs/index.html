Example usage

Install yurbacli

npm install yurbacli

Create new simple bot and registe command

// bot.js
const { Client } = require('yurbacli');

const bot = new Client('TOKEN', '!');

bot.registerCommand('ping', {}, async (msg) => {
	msg.reply('pong');
});

bot.init();

    About

    This repository contains multiple packages that work with the Yurba.one API and are required for the main package to function. The main package is Yurba.js.

    This library draws inspiration from discord.js, so you may notice some similarities in design and structure.

    Packages

    • yurba.js (source) - The powerful library for creating bots and integrating with the Yurba API.
    • @yurbajs/rest (source) - REST client for Yurba API.
    • @yurbajs/ws (source) - WebSocket client for yurba.one.
    • @yurbajs/types (source) - Types for yurba.js packages and yurba.one.
